Как помочь?
Мы рады контрибьюторам любого уровня! Начать легко:

Чек-лист нового контрибьюта:
1. Сделайте форк репозитория (кнопка Fork в правом верхнем углу).
2. Клонируйте свой форк локально:
     git clone https://github.com/ВАШ-ЛОГИН/Maksimova-Lyubov-mgutu-fct.git
3. Перейдите в папку проекта:
     cd Maksimova-Lyubov-mgutu-fct
4. Установите зависимости:
     pip install -r requirements.txt
5. Запустите тесты:
     python -m pytest tests/
6. Внесите правки -> заификсируйте -> отправьте PR.
